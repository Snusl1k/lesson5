GIT flow

GITclone - для того, щоб "склонить" репозиторий
GITpull - для того, щоб отримати зміни з репозиторію
GITcheckout -b branchname - створити нову вітку та переключитися на неї
GITadd . - добавить изменения, которие сделали локально
GITcommit -m "commit text" - подписать их
GITpush origin branchname - запушить на git сервер
